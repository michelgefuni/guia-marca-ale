Guia de Marca ALE
------------------------------------

### Como iniciar

```sass --watch scss/main.scss css/style.css
```

### Estrutura de pasta
```
├── index.html
├── css
│   ├── style.css (gerado automaticamente)
├── scss
|   ├── main.scss
│   ├── base
│       ├── _colors.scss
│       ├── _default.scss
│       ├── _fonts.scss
│   ├── layout
│       ├── _Header_.scss
│       ├── _Hero.scss
├── js
│   ├── main.js
├── assets
│   ├── icons
│   ├── images
├── pages
```
